# Project Brief

## Overview
The AWS EKS Secure Delivery & Runtime Detection Lab is a purpose-built environment designed to demonstrate an understanding of DevSecOps, Cloud Security Posture Management (CSPM), and Runtime Threat Detection.

## Objective
To prove that a developer cannot push insecure infrastructure, vulnerable container images, or dangerous Kubernetes workloads into a production-like environment without being blocked or triggering a documented incident response workflow.

## Target Audience
This repository serves as a portfolio piece for Cloud Security Engineers, DevSecOps Engineers, and SOC Analysts, showcasing end-to-end security architecture.
