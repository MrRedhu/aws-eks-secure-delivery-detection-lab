# Project Brief

## Overview
The AWS EKS Secure Delivery and Runtime Detection Lab is a deployed, evidence-backed portfolio project for DevSecOps, Cloud Security Posture Management, Kubernetes hardening, and runtime detection.

## Objective
Prove that insecure infrastructure, vulnerable images, and dangerous Kubernetes workloads can be blocked before deployment while runtime threats are detected and routed through an incident response path.

## Target Audience
This repository serves as a portfolio piece for Cloud Security Engineers, DevSecOps Engineers, and SOC Analysts, showcasing end-to-end security architecture.

## Current Status
The dev environment has been provisioned, the demo app has been deployed to EKS, Kyverno policies are active, and validation evidence is captured in `EVIDENCE_SUMMARY.md`.
