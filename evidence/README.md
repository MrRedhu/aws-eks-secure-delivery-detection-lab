# Evidence Folder

Use this folder for sanitized validation artifacts that support the portfolio case study.

Suggested evidence:

- Checkov before/after scan output
- Trivy before/after scan output
- Kyverno policy pass/fail output
- GuardDuty finding examples
- EventBridge to Lambda to SNS routing proof
- Security Hub control screenshots
- Runbook walkthrough notes

Do not commit AWS secrets, kubeconfig files, raw Terraform state, full account inventories, or unredacted personal data.
